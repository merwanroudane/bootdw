Quick Start Guide

This guide will help you get started with boot_dw in just a few minutes.

Basic Workflow

Testing for autocorrelation in regression models using boot_dw follows these simple steps:

  1. Import the necessary functions
  2. Prepare your data (dependent and independent variables)
  3. Choose a test method
  4. Interpret the results

Simple Example

Here's a minimal example to test for autocorrelation:

import numpy as np
from boot_dw import bca_rho_test

# Generate sample data
np.random.seed(42)
n = 100

# Independent variables
X = np.random.randn(n, 2)

# Generate autocorrelated errors
rho_true = 0.7
errors = np.zeros(n)
errors[0] = np.random.randn()
for t in range(1, n):
    errors[t] = rho_true * errors[t-1] + np.random.randn()

# Dependent variable with autocorrelated errors
y = 2 + 3*X[:, 0] - 1.5*X[:, 1] + errors

# Run the BCa-œÅ test (most powerful method)
result = bca_rho_test(y, X, random_state=42)
print(result)

Available Tests

boot_dw provides four different tests for autocorrelation:

1. Classical Durbin-Watson Test

from boot_dw import dw_test

result = dw_test(y, X)
print(result)

Note: The classical DW test has an indeterminate range and doesn't provide exact p-values. Consider using one of the bootstrap tests instead.

2. Bootstrapped Durbin-Watson (BDW) Test

from boot_dw import bdw_test

result = bdw_test(
    y, X,
    n_bootstrap=200,  # Number of bootstrap replications
    alpha=0.05,       # Significance level
    random_state=42   # For reproducibility
)
print(result)

The BDW test eliminates the indeterminate range and provides exact p-values.

3. Bootstrapped œÅ (B-œÅ) Test

from boot_dw import b_rho_test

result = b_rho_test(
    y, X,
    n_bootstrap=200,
    alpha=0.05,
    random_state=42
)
print(result)

The B-œÅ test directly bootstraps the autocorrelation coefficient.

4. BCa-œÅ Test (Recommended)

from boot_dw import bca_rho_test

result = bca_rho_test(
    y, X,
    n_bootstrap=200,
    alpha=0.05,
    random_state=42
)
print(result)

Tip: The BCa-œÅ test (Bias-Corrected Accelerated œÅ) has the best small sample properties and is the most powerful test. This is the recommended method for most applications.

Understanding the Results

All test functions return a TestResult object with the following attributes:

# Access test results
print(f"Test statistic: {result.statistic}")
print(f"P-value: {result.pvalue}")
print(f"Method: {result.method}")
print(f"Alternative: {result.alternative}")

# Additional information
for key, value in result.additional_info.items():
    print(f"{key}: {value}")

Interpreting P-values

  • p-value < 0.05: Reject H‚ÇÄ, evidence of autocorrelation
  • p-value ‚â• 0.05: Fail to reject H‚ÇÄ, no strong evidence of autocorrelation

One-Line Testing

For quick analysis, use the unified autocorrelation_test function:

from boot_dw import autocorrelation_test

# Use the best method (BCa-œÅ) by default
result = autocorrelation_test(y, X, random_state=42)

# Or specify a different method
result = autocorrelation_test(
    y, X,
    method='bdw',  # Options: 'dw', 'bdw', 'b_rho', 'bca_rho'
    n_bootstrap=200,
    random_state=42
)
print(result)

Testing Alternatives

By default, all tests test H‚ÇÄ: œÅ = 0 against H‚ÇÅ: œÅ > 0 (positive autocorrelation). You can change this using the alternative parameter:

# Test for positive autocorrelation (default)
result = bca_rho_test(y, X, alternative='greater')

# Test for negative autocorrelation
result = bca_rho_test(y, X, alternative='less')

# Two-sided test
result = bca_rho_test(y, X, alternative='two-sided')

Common Use Cases

Time Series Regression

Autocorrelation is common in time series data:

from boot_dw import bca_rho_test
import numpy as np

# Time series data
years = np.arange(1990, 2024)
gdp_growth = np.array([...])  # Your GDP growth data
inflation = np.array([...])    # Your inflation data

# Test for autocorrelation
result = bca_rho_test(gdp_growth, inflation.reshape(-1, 1))
print(result)

Panel Data

For panel data, test autocorrelation within each entity:

from boot_dw import bca_rho_test

# Loop through each entity
for country in countries:
    # Extract data for this country
    y_country = y[country_id == country]
    X_country = X[country_id == country]
    
    # Test for autocorrelation
    result = bca_rho_test(y_country, X_country)
    print(f"{country}: p-value = {result.pvalue:.4f}")

Bootstrap Parameters

The number of bootstrap replications affects both accuracy and computation time:

n_bootstrap Use Case Computation Time
100 Quick exploratory analysis Fast
200 Standard analysis (recommended) Moderate
1000+ Publication-quality results Slower

Tip: Start with n_bootstrap=200 for standard analysis. This matches the value used in Jeong & Chung (2001) and provides good balance between accuracy and speed.
