Installation

Requirements

Boot_dw requires Python 3.7 or higher. The package depends on the following libraries:

  • numpy - For numerical computations
  • scipy - For statistical distributions

Install via pip

The easiest way to install boot_dw is using pip:

pip install boot-dw

Install from source

You can also install boot_dw directly from the source code:

# Clone the repository
git clone https://github.com/your-repo/boot_dw.git
cd boot_dw

# Install in development mode
pip install -e .

Verify Installation

To verify that boot_dw has been installed correctly, run the following Python code:

import boot_dw

print(f"boot_dw version: {boot_dw.__version__}")
print(f"Available functions: {boot_dw.__all__}")

You should see output similar to:

boot_dw version: 0.0.1
Available functions: ['durbin_watson', 'ols_regression', 'dw_test', 'bdw_test', 
                      'b_rho_test', 'bca_rho_test', 'autocorrelation_test', ...]

Optional Dependencies

For running examples and working with real data, you may want to install additional packages:

# For data manipulation and visualization
pip install pandas matplotlib seaborn

# For downloading economic data from FRED
pip install pandas-datareader fredapi

Development Installation

If you want to contribute to boot_dw development, install the development dependencies:

pip install boot-dw[dev]

This will install additional tools for testing and development:

  • pytest - For running tests
  • black - For code formatting
  • flake8 - For code linting
  • mypy - For type checking

Troubleshooting

ImportError: No module named 'boot_dw'

If you get this error, ensure that:

  1. You've installed the package using pip install boot-dw
  2. You're using the correct Python environment
  3. The package is installed in the same environment where you're running Python

NumPy/SciPy Installation Issues

On some systems, you may need to install NumPy and SciPy before installing boot_dw:

pip install numpy scipy
pip install boot-dw

Permission Denied

If you encounter permission errors on Linux/Mac, use:

pip install --user boot-dw

Note: It's recommended to install boot_dw in a virtual environment to avoid conflicts with other packages.

Creating a Virtual Environment

Using venv (Python 3.3+)

# Create a virtual environment
python -m venv myenv

# Activate the virtual environment
# On Windows:
myenv\Scripts\activate
# On Linux/Mac:
source myenv/bin/activate

# Install boot_dw
pip install boot-dw

Using conda

# Create a conda environment
conda create -n myenv python=3.9

# Activate the environment
conda activate myenv

# Install boot_dw
pip install boot-dw
